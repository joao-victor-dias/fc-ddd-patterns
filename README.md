## FC-DDD-PATTERNS

### Descrição

Este projeto faz parte de um desafio do curso Full Cycle.

O objetivo é aplicar conceitos de **Domain Driven Design (DDD)** e **Test Driven Development (TDD)**, utilizando como base um template previamente fornecido pelo curso.

---

### Tecnologias utilizadas

- TypeScript
- Node.js
- Jest
- Sequelize
- SQLite (in-memory)

---

### Como executar o projeto

#### Instalar dependências

```bash
npm install
```
#### Rodar testes

```bash
npm test
```
